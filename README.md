# Product-Management-System-CRUDS-
A simple yet powerful web application that allows users to create, read, update, and delete (CRUD) products efficiently.
It features a clean interface, local storage integration, real-time total calculation, and search functionality by both title and category.

#Features

Add, update, and delete products easily

Save data locally using localStorage

Search products by title or category

Responsive design for all devices

Real-time total price calculation including taxes, ads, and discount

Delete all products with one click

Built using pure JavaScript (no frameworks)

#Technologies Used

HTML5 – Structure and layout

CSS3 – Styling and responsive design

JavaScript (ES6) – Application logic and CRUD operations

LocalStorage API – Data persistence
